# dev-env

## 構築手順

### リポジトリのクローン

git clone https://

### edgematrix-webのクローン

- `project`ディレクトリ配下に、webのクローン

cd project<br>
git clone https://

### dockerの立ち上げ

- `docker`ディレクトリに移動し、dockerの立ち上げ

cd docker<br>
docker-compose up -d

### アクセス

- 以下にアクセスして、トップページが表示されれば、構築成功です


http://localhost:8001/ <br>
http://localhost:8002/
